# project-scheduling-heuristics
This repository contains clear and minimal implementations of classical heuristics for project and 2-machine scheduling problems, with a focus on makespan minimization

F2||Cmax : algorithme jonhson 1956(gives optimlae solution)
J2||Cmax : algorithme jackson  1956(gives optimal solution)
O2||Cmax : algorithems LAPT,GARD,Gonzalez (1976)
